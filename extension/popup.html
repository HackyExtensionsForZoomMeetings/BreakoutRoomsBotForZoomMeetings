Breakout Room Bot for Zoom Meetings

Allows Attendees to switch their Breakout Rooms via Group Chat.

If you would like to modify or change the logic for this bot for your own needs, please check out the Source Code for this Chrome extension, modify it, uninstall this extension, and load your modified extension for your meeting's needs and purposes.

Please read the README for usage instructions.

⚠️ A Host or Co-Host session will need to be dedicated to the meeting with this bot to manage Breakout Rooms! Audio participation in bot-hosting session not required. Join with another client on the system to actually participate.

⚠️ Ensure your view of of the Zoom Meeting has the Breakout Rooms window IN PROGRESS with a RED "Close All Rooms button" visible. Example image of Window on left.

Once that is done, press the button below to attach and launch the bot.

To remove or deactivate the bot, refresh or close the meeting page.

KNOWN ISSUE: Do not send bot commands from attached host meeting! Have an attendee send a !ls command to test.
