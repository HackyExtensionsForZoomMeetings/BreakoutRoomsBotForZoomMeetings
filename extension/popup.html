Breakout Rooms Bot for Zoom Meetings

Allows Attendees to switch their Breakout Rooms via Group Chat or Renames.

Bot only runs on Zoom Meetings Web Client ("Join from Browser"). Attendees can run whatever client they like!

If you would like to modify or change the logic for this bot for your own needs, please check out the Source Code for this Chrome extension, modify it, uninstall this extension, and load your modified extension for your meeting's needs and purposes.

Please read the README for usage instructions.

⚠️ The Host Zoom Meetings web client in the web browser will need to be dedicated to the meeting with this bot to manage Breakout Rooms! Audio participation in bot-hosting session not required. Do not join breakout rooms with this client! Host, join with another Zoom Meetings client on your system to actually participate in the meeting and join breakout rooms.

⚠️ Ensure your view of of the Zoom Meeting has the Breakout Rooms window IN PROGRESS with a RED "Close All Rooms button" visible. Example image of Breakout Rooms window in correct state on left. Do not close the Chat window while the bot is in operation. You may close the breakout rooms window though.

Once that is done, press the button below once to attach and launch the bot.

To remove or deactivate the bot, refresh or close the meeting page.

KNOWN ISSUE: Do not send bot commands from attached host meeting! Have an attendee send a !ls command to test.
